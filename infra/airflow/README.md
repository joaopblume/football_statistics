# Apache Airflow

For this project, Apache Airflow should be installed directly on the operating system (OS), rather than running inside a Docker container.

## Installation

Please follow the official Apache Airflow documentation to install and configure the orchestrator on your specific operating system:

[Official Apache Airflow Installation Guide](https://airflow.apache.org/docs/apache-airflow/stable/installation/index.html)

Make sure to set up the appropriate Python environment and initialize the Airflow database after installation.
